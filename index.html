Projects

Analysis and Implementation of Evolutionary Algorithms in Computer Games to build smarter non-playable characters

Assistant Professor : Dr. Chitra Dadkhah

Non Playable Characters (NPCs) have a significant role in making game more enjoyable and amusing. Therefore by improving them and making them smarter we can produce a greate effect on player. One way to improve NPCs is to generate them as campatible with environment as they can be. for example trucks with different tire sizes can travel different distances. By using evolutionary algorithms we can reach the best physical and behavioral specification for NPCs, for example best size of tire for a truck. I am trying to implement and evaluate different evolutionary algorithms for generating smarter and stronger NPCs as my bachelor thesis.

Development of Second-fastest solver for slitherlink puzzle using Monte Carlo Tree Search Methodes

Assistant Professor : Dr. Dara rahmati
During my internship in the High Performance Computing (HPC) department at IPM, I was in charge of studying, implementing, adopting, and evaluating the Monte Carlo Tree Search algorithm (MCTS) on the Slitherlink puzzle. This was quite a challenging task and involved a lot of effort. After months of dedication, working with fellow researchers, we managed to build the fastest solver for five puzzles including Slitherlink using MCTS. We prepared a publication on our work and are currently submitting it to be reviewed for the IEEE Transaction on Games journal
